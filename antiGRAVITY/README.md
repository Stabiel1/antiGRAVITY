
## Quick Start

```bash
# Run the interactive App Builder
python main.py

# Or create a project directly
python main.py create "build a REST API for task management"
python main.py create "make a web dashboard"
python main.py create "build a CLI tool for file organization"
python main.py create "build an AI agent for news monitoring"

# Check Trinity status
python main.py status
```
